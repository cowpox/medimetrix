# Medimetrix Frontend

This is a Vite + React + TypeScript scaffold for the Medimetrix project.

## Available Scripts

- `npm install` – install dependencies.
- `npm run dev` – start the development server with hot module replacement.
- `npm run build` – type-check and generate a production build.
- `npm run preview` – locally preview the production build.

## Project Structure

```
front/
├── index.html
├── package.json
├── src/
│   ├── App.tsx
│   ├── main.tsx
│   └── styles.css
└── vite.config.ts
```

Feel free to customize the scaffold to match your product requirements.
